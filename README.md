# Redux Playground

A simple Redux project for introduction purpose.

## Installation

Install redux-playground with npm

```bash
  npm -g json-server
  npm install
```

Type this command to run the project

```bash
  npm run dev
```

Open 1 more terminal and type this command

```bash
  json-server -w db.json
```
